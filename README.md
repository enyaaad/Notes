# posts

Project isnt builded. To use it, you must to create file:

```sh
env.local
```

And paste API URL to VITE_API_URL variable

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```


### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
